## 概述

S-Protect-PY 将 Python 项目加密为独立可运行的输出，保护源码不被逆向——无论是人类分析还是 AI 大模型分析。

### 核心能力

- **AST 混淆 L5** — 变量/函数/类重命名、字符串分散、不透明表达式注入、控制流平坦化(if/elif + match/case)、隐式数据流、死代码注入
- **反 AI 逆向** — 语义噪声 5x(200-300诱饵)、上下文污染、对抗性加密库导入
- **多层加密** — 6 层 HKDF/AES-GCM + 层间哈希链验证(防逐层提取)
- **远程密钥分片** — 支持 HTTPS 直连远程密钥服务器，离线不可解
- **反模拟器** — 检测 QEMU/VMware/VirtualBox/KVM 等分析环境
- **内存自删除** — .pye 模块解密后立即擦除源文件
- **文件系统陷阱** — 30% 诱饵 .pye 加载即触发反制
- **调用栈混淆** — 6 层深 lambda 栈噪声
- **反调试/反沙箱** — 10 种检测 + 进程枚举 + 时间戳反演
- **双进程架构** — 父进程持密钥、子进程执行业务(实验性)
- **Shamir 密钥分片** — GF(256) Lagrange 插值恢复 master_key
- **代码虚拟化** — 自定义栈式 VM，编译 Python 函数为 bytecode 执行
- **数字水印** — 三层(文件/代码/运行时)溯源追踪
- **NTP 过期控制** — 联网校验 + 时钟偏移检测
- **混合加密** — RSA/ECC 公钥加密 master_key
